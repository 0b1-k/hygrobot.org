HyGroBot V2 Build

This timelapse shows how the second HyGroBot prototype was built in a few days, in preparation for Hempfest 2013 where we'll be showing it at our booth (#482). This build draws from all the experience that we gained from building the first HyGroBot prototype and the lessons that we learned along the way. In this version, we mostly used readily available parts found at local hardware and gardening stores, some of our own electronics and enclosures available on Nwazet.com, a Black Dog LED lamp and a few new products that we'll release later this month. This HyGroBot prototype uses a Raspberry Pi computer running Linux (Raspbian) to drive sensors and actuators, while another Raspberry Pi is dedicated to capturing pictures and, eventually, video. These two functions will be consolidated into a single unit in the production version of HyGroBot coming Fall 2013. The software is entirely written in Python as well. Unlike its predecessor, this second HyGroBot prototype is not feeding its data to the Internet by default: we're designing it so that it can be used 'off-the-grid', with its own web interface and data collection system.

Some important aspects of plant growth such as air filtration, moisture control, water and air heating and cooling, dispensing carbon dioxide, preventing light and air leaks, etc. will not be covered in this prototype build. They will be addressed in the commercial version of HyGroBot.

We will soon publish the schematics, source code and bill of materials used in this prototype. Follow the project to be notified.
